# 🚀 HELIOS - Quick Start Guide

## Get Started in 3 Steps

### Step 1: Open HELIOS
- Double-click `index.html` or drag it into Chrome/Edge
- Modern browsers recommended (Chrome 90+, Edge 90+)

### Step 2: Grant Permissions
When prompted, allow:
- ✅ **Camera** (for hand tracking)
- ✅ **Microphone** (for voice control)

### Step 3: Start Using
You're in! HELIOS is now active.

---

## 🎯 First Actions

### Try Voice Commands:
Say these out loud:
- **"Open YouTube"** - Opens YouTube in a floating window
- **"Search for AI news"** - Google search appears
- **"Next tab"** - Switch between windows

### Try Gestures:
- ✋ **Point your index finger** - See the cursor move
- 👌 **Pinch thumb + index** - Click/select
- 👈 **Swipe left/right** - Switch tabs
- 👆 **Swipe up** - Show all windows

### Try Keyboard:
- `Ctrl + T` - New window
- `Tab` - Next window
- `Ctrl + Space` - Overview mode

---

## 💡 Tips

1. **Speak clearly** - Moderate volume, clear pronunciation
2. **Good lighting** - Helps hand tracking
3. **Center yourself** - Face the camera directly
4. **One hand** - Right hand preferred for gestures
5. **Wait for feedback** - AI core pulses when listening

---

## 🎨 What You're Seeing

- **Blue sphere in center** = AI Core (responds to your voice)
- **Floating particles** = Ambient environment
- **Cyan cursor** = Your hand position
- **Floating windows** = Real websites in 3D space
- **Top bar** = Voice status and system info

---

## 🔊 Audio Feedback

Listen for:
- **Beep (rising)** = Command confirmed
- **Beep (falling)** = Error/not recognized
- **Click** = Interaction registered
- **Ambient hum** = System active

---

## ⚠️ Common Issues

**Camera not working?**
→ Check permissions, close other apps using camera

**Voice not working?**
→ Check microphone permissions, try refreshing

**Slow performance?**
→ Close other tabs, reduce window count

**Can't see hand cursor?**
→ Ensure good lighting, center your hand in frame
