# 🎮 HELIOS Interactive Earth - Updated Controls

## ✅ Control Scheme (Updated: Feb 12, 2026)

### **ONE HAND**

#### **🤏 Pinch (Thumb + Index)**
- Pick up Earth and move it around
- Release pinch to drop (gravity + bounce)

#### **✊ Fist/Grab**
- Alternative to pinch - same function
- Make a fist to pick up Earth
- Open hand to drop

### **TWO HANDS**

#### **🤏🤏 Pinch BOTH Hands + Move**
- **Rotation mode** - Move both pinched hands to rotate Earth
- Horizontal movement → Horizontal rotation
- Vertical movement → Vertical rotation
- **500x sensitivity** - tiny movements = big rotation!

#### **🙌 Open Hands (Not Pinching)**
- No action (previously was rotation)
- HUD shows: "two hands (not pinching)"

5. **Bright lighting** - Much easier to see terrain details
